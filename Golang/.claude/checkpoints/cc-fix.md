# cc-fix Checkpoints

## 开始前检查

| 检查项 | 结果 |
|--------|------|
| [ ] review 结果已读 | |
| [ ] 问题清单已记录 | |
| [ ] `review.md` 已存在 | |
| [ ] 本轮需要处理的问题范围已确认 | |

## 执行中检查

| 检查项 | 结果 |
|--------|------|
| [ ] 默认只处理 `status = open` 的 Findings | |
| [ ] 若临时追加问题，已先写入 `review.md` 或 `log.md` | |
| [ ] 修复代码时已同步更新 `spec.md` / `tasks.md` / `log.md` / `review.md` | |
| [ ] 每项修复后重新验证，且验证等级未低于既有要求 | |

## 完成后检查

| 检查项 | 结果 |
|--------|------|
| [ ] 已修复问题改为 `fixed`，未修复问题仍保留为 `open` | |
| [ ] `go build ./...` 通过 | |
| [ ] 未清空原有 review 结论 | |
